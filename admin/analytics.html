Configure Embedded Report

How to get your embed URL:

  1. Go to Looker Studio (formerly Data Studio)
  2. Create a new report or use a template connected to your GA4
  3. Click Share > Embed report
  4. Enable embedding and copy the embed URL
  5. Paste the URL below

The URL should start with https://lookerstudio.google.com/embed/

Setting Up Google Analytics

1. Create GA4 Property

  • Go to Google Analytics
  • Click Admin > Create Property
  • Follow the setup wizard

2. Add Tracking Code

  • Get your Measurement ID (G-XXXXXXX)
  • Add the gtag.js snippet to your site's <head>
  • Verify data is being collected

3. Create Looker Studio Report

  • Go to Looker Studio
  • Use a GA4 template or create custom
  • Connect to your GA4 property

4. Embed in Admin

  • Share your Looker Studio report
  • Enable embedding
  • Paste the embed URL above
