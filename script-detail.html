Batch Export Pages

Export multiple InDesign pages to individual PDF files with custom naming conventions, compression settings, and preset configurations. Perfect for automating repetitive export tasks.

About This Script

Batch Export Pages is a powerful InDesign script that automates the process of exporting multiple pages to individual PDF files. Whether you're working on a catalog with hundreds of pages or need to distribute individual sheets to different clients, this script saves you hours of manual work.

Key Features

  • Export all pages or a custom page range to individual PDFs
  • Customizable file naming with variables (page number, document name, date)
  • Support for all InDesign PDF presets
  • Option to include bleed and slug areas
  • Automatic folder creation for organized output
  • Progress bar with estimated time remaining
  • Error logging and summary report

Use Cases

  • Catalog production - export each product page separately
  • Magazine publishing - distribute individual articles
  • Marketing materials - create separate PDFs for each design variation
  • Print production - prepare individual sheets for press

System Requirements

  • Adobe InDesign CC 2019 or later
  • Windows 10/11 or macOS 10.14+
  • Minimum 4GB RAM (8GB recommended)

Installation Instructions

The installation process is the same for both Windows and macOS. The script will work immediately after placing it in the correct folder.

Step 1: Download the Script

Click the download button to save the BatchExportPages.jsx file to your computer.

Step 2: Locate the Scripts Folder

Navigate to the InDesign Scripts Panel folder:

  • Windows: C:\Users\[Username]\AppData\Roaming\Adobe\InDesign\[Version]\en_US\Scripts\Scripts Panel\
  • macOS: ~/Library/Preferences/Adobe InDesign/[Version]/en_US/Scripts/Scripts Panel/

Tip: In InDesign, go to Window > Utilities > Scripts, then right-click on "User" and choose "Reveal in Explorer/Finder" to quickly locate the folder.

Step 3: Copy the Script

Copy the downloaded BatchExportPages.jsx file into the Scripts Panel folder.

Step 4: Verify Installation

Open InDesign and go to Window > Utilities > Scripts. You should see "BatchExportPages" in the User folder. Double-click to run!

How to Use

Getting Started

  1. Open your InDesign document
  2. Go to Window > Utilities > Scripts
  3. Double-click on "BatchExportPages" in the User folder
  4. The script dialog will appear with export options

Configuration Options

Page Range

Choose "All Pages" to export every page, or "Range" to specify custom pages (e.g., "1-5, 8, 12-15").

File Naming

Use variables to create custom file names:

  • {docname} - Document name
  • {page} - Page number
  • {date} - Current date
PDF Preset

Select from your installed PDF presets including [High Quality Print], [Press Quality], or any custom presets you've created.

Output Folder

Click "Browse" to select where exported PDFs should be saved. Optionally create subfolders automatically.

Tips & Best Practices

  • Save your document before running the script
  • For large documents, close other applications to free up RAM
  • Use descriptive file naming patterns for easier organization
  • Test with a few pages first before exporting the entire document
